# Береста

Береста — форк [OpenPencil](https://github.com/open-pencil/open-pencil), открытого редактора дизайна, адаптированный под нужды дизайн-систем.

Открывает `.fig` и `.pen` файлы. Работает в браузере и как десктопное приложение (Tauri v2).

## Отличия от OpenPencil

- **Именованные стили** — библиотека FillStyle, TextStyle, EffectStyle с привязкой к нодам и undo-поддержкой
- **Дизайн-библиотеки** — импорт `.fig`/`.pen` файлов как общих библиотек: компоненты, переменные и стили доступны во всех документах; не сохраняются в `.fig` экспорт
- **Линтер** — панель проверки качества с 17 правилами и тремя пресетами (recommended / strict / accessibility); автозапуск при изменениях, клик на ошибку выделяет ноду
- **Экспорт токенов** — выгрузка переменных и стилей в CSS custom properties, Tailwind theme config или W3C Design Tokens JSON

## Запуск

```sh
bun install
bun run dev
```
